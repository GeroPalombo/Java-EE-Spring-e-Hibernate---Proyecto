# Java-EE-Spring-e-Hibernate---Proyecto
Proyecto para practicar fundamentos de Spring e Hibernate en Java EE, utilizando Eclipse como IDE. El proyecto consta de una agenda de reservas hoteleras que se gestiona en el navegador a traves del puerto 80, donde se almacenan datos del cliente, tales como nombres, emails, tel, fecha de entrada y salida del establecimiento, etc. 
Para el desarrollo del proyecto se usaron herramientas como maven, docker y omnidb en lo que es bases de datos, hibernate para la capa de persistencia de datos, Spring y algunas de sus dependencias como SpringBoot, Spring data JPA, Spring Rest y Swagger y en menor medida, Spring Segurity, que se utilizo para agregar una pantalla principal donde se debe colocar usario y contrase√±a para acceder a la agenda.
